# mo — Markdown Viewer Skill

moを使って、Claude Codeが出力したMarkdownファイルをターミナル上でレンダリングして閲覧するスキルです。

## 実行フロー

### 1. moのインストール確認

`which mo` を実行し、moがインストールされているか確認する。

- インストールされていない場合 → 「moがインストールされていないためスキップします」と伝えて終了する
- インストールされている場合 → 次のステップへ進む

### 2. 対象ファイルの特定

以下の優先順位で対象ファイルを決定する：

1. ユーザーが明示的にファイルパスを指定した場合 → そのパスを使用する
2. 現在の会話内で直近に出力・作成したMarkdownファイルがある場合 → そのファイルを使用する
3. どちらも該当しない場合 → ユーザーにファイルパスを確認する

### 3. ファイルの存在確認

対象ファイルが実際に存在するか確認する。存在しない場合はユーザーに報告する。

### 4. moで開く

以下のコマンドでMarkdownファイルをmoで開く：

```bash
mo <file_path>
```

バックグラウンドで実行し、ユーザーに開いた旨を報告する。

### 5. 報告

「moで `<file_path>` を開きました」と簡潔に報告する。
