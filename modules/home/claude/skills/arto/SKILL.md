# Arto — Markdown Viewer Skill

artoを使って、Claude Codeが出力したMarkdownファイルをGitHub風にレンダリングして閲覧するスキルです。

## 実行フロー

### 1. artoのインストール確認

`which arto` を実行し、artoがインストールされているか確認する。

- インストールされていない場合 → 「artoがインストールされていないためスキップします」と伝えて終了する
- インストールされている場合 → 次のステップへ進む

### 2. 対象ファイルの特定

以下の優先順位で対象ファイルを決定する：

1. ユーザーが明示的にファイルパスを指定した場合 → そのパスを使用する
2. 現在の会話内で直近に出力・作成したMarkdownファイルがある場合 → そのファイルを使用する
3. どちらも該当しない場合 → ユーザーにファイルパスを確認する

### 3. ファイルの存在確認

対象ファイルが実際に存在するか確認する。存在しない場合はユーザーに報告する。

### 4. artoで開く

以下のコマンドでMarkdownファイルをartoで開く：

```bash
arto <file_path>
```

バックグラウンドで実行し、ユーザーに開いた旨を報告する。

### 5. 報告

「artoで `<file_path>` を開きました」と簡潔に報告する。
